### UPDATE
* Utilizado para modificar datos de una tabla.
* Syntaix
```
UPDATE Nombre_Tabla
SET ([Nombre_Columna]) = [Value]
WHERE [Condición]
```
* Queremos modificar Firstname y Lastname de esta tabla:
<img width="669" height="271" alt="image" src="https://github.com/user-attachments/assets/d391f5fc-bd8e-480a-b5f9-46cbdc8fb373" />

```
UPDATE AUTHOR
SET LASTNAME = 'KATTA', FIRSTNAME = 'LAKSHMI'
WHERE AUTHOR_ID = 'A2'
```

### DELETE
* Utilizado para modificar una o más filas de una tabla.
* Syntaxis:
```
DELETE FROM Nombre_Tabla
  WHERE [Condición]
```
* Queremos eliminar filas A2 y A3:
<img width="608" height="154" alt="image" src="https://github.com/user-attachments/assets/cb86bb44-05ba-4203-986c-0eff6658f371" />
```
DELETE FROM AUTHOR
WHERE Author_ID IN ('A2', 'A3')
```
