### COUNT
* Funcion incluída para manejar bases de datos.
* Devuelve cantidad de filas.
* Contar todas las filas de la tabla:
```
SELECT COUNT(*) FROM Nombre_Tabla
```
* Consdieremos la siguiente tabla "MEDALS":
<img width="467" height="191" alt="image" src="https://github.com/user-attachments/assets/ebad23d5-8944-4ff1-a10f-acf3878e2542" />

* Queremos contar cuántas veces el ganador de la medalla era de Canada:
```
SELECT COUNT(COUNTRY) FROM MEDALS WHERE COUNTRY = "CANADA"
```

### DISTINCT
* Utilizado para remover duplicados del set resutlado.
* Útil si queremos obtener filas con valores únicos en las columnas: 
```
SELECT DISTINCT Nombre_Columna FROM Nombre_Tabla
```
* Consideremos la misma tabla que en el ejemplo anterior, pero ahora queremos obtener los países que obtuvieron medallas
  de ORO. Ojo que un país puede haber obtenido más de una:
```
SELECT DISTINCT(COUNTRY) FROM MEDALS WHERE MEDALTYPE = "GOLD"
```

### LIMIT
* Restringe el número de filas que obtenemos de la base de datos.
* Por ejemplo si queremos solamente las primeras 10 filas:
```
SELECT * FROM Nombre_Tabla LIMIT 10
```
* Otro ejemplo, si quisieramos 5 filas de la tabla medals de un año en específico (2019):
```
SELECT * FROM MEDALS
  WHERE YEAR = 2018 LIMIT 5
```
* Queremos contar 15 filas de la tabla, comenznado desde la fila 11:
```
SELECT * from MEDALS LIMIT 15 OFFSET 10;
```
