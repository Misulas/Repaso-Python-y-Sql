### SELECT
* Lo utilizamos para ver los datos que se encuentran en las tablas.
* Es un DML.
* Forma más simple:
  * SELECT * FROM "nombre_tabla".
*  Consideremos esta tabla de la entidad "Book", la cual contiene los atributos de los libros:
<img width="429" height="128" alt="image" src="https://github.com/user-attachments/assets/f890b941-f6c7-465d-9b98-8b8adab2ab48" />

* Si hacemos: 

```
SELECT * FROM Book
```

<img width="649" height="152" alt="image" src="https://github.com/user-attachments/assets/72304454-2a38-4086-bfa9-1e84d3c3fff9" />

* Por ahora hemos utilizado "*" para seleccionar lo que queremos, esto lo podemos reemplazar por el nombre de la columna que necesitamos
* Ejemplo: El siguiente código despliega solamente las columnas "Book_ID" y "Title": 
```
SELECT Book_ID, Title from Book
```

### WHERE
* Restringe el set resultado.
* Condición que evalúa: True, False o Unkown.
* Utilizado en la condición de búsqueda.
* El siguiente código nos entrega Los ids de los libros y los títulos (en formato de tabla), condicionado porque el "Book_ID" sea igual
  a B1.
  
```
 SELECT Book_ID, Title FROM BOOK
    WHERE Book_ID='B1' 
```
