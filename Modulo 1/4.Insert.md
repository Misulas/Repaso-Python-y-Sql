### CREATE
* Utilizado para crear una tabla.
* Syntaxis de ejemplo:
```
CREATE TABLE nombre_tabla (
  columna1 datatype [constraints],    por ejemplo: NOT NULL
  columna2 datatype [constraints],
  ...
  columnaN datatype [constraints],
  [PRIMARY KEY (nombre_columna(s))],
  [FOREGIN KEY (nombre_columna(s))],
  [UNIQUE (nombre_columna(s))],
  [CHECK (condition)]

Ejemplo concreto:
CREATE TABLE Customers (
  CustomerID INT PRIMARY KEY,
  FirstName VARCHAR(50) NOT NULL,
  LastName VARCHAR(50) NOT NULL,
  Email VARCHAR(100) UNIQUE,
  Age INT CHECK (Age >= 18),
  RegistrationDate DATE DEFAULT CURRENT_DATE
```

### INSERT
* Utilizado para poblar una tabla. Insertando filas a una tabla.
* Data Manipulation Language (DML): Utilizado para leer y modificar datos.
* Syntaxis de ejemplo:
```
INSERT INTO nombre_tabla
  ([Nombre_Columna], [Nombre_Columna2],...)
VALUES
  ([Valor1], [Valor2],...)

Ejemplo concreto:
INSERT INTO AUTHOR
  (AUTHOR_ID, LASTNAME, FIRSTNAME, EMAIL, CITY, COUNTRY)
VALUES
  ('A1', 'Chong', 'Raul', 'raul@gmail.com', 'Toronto', 'CA')
  ('A2', 'Ahuja', 'Rav', 'lol@gmail.com', 'Toronto', 'CA')
```
