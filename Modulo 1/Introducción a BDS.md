## ¿Qué es SQL?
* Es un lenguaje utilizado para bases de datos relacionales, con el fin de obtener datos desde estas.

## ¿Qué son los datos? 
* Colección de hechos en forma de palabras, números o imágenes.
* Crucial para cualquier negocio.
* Necesitan estár seguros.

## ¿Qué son las bases de datos?
* Repositorio de datos; programa que guarda datos.
* Entrega funcionalidad de hacer CRUD a los datos.
* Se puede guardar los datos en diferentes formatos.
* Cuando los datos están tabulados, se organizan en tablas; Base da datos relacional.

## DBMS: Database Management System 
* Herramientas de software para controlar los datos en la DB.

## RDBMS: Relational Database Management System
* MySQL, Oracle Database, IBM, Db2, etc.

## Comandos SQL básicos:
* Crear una tabla -> CREATE.
* Insertar un dato -> INSERT.
* Seleccionar datos de las tablas -> SELECT.
* Actualizar datos de las tablas -> UPDATE.
* Eliminar datos de las tablas -> DELETE.
